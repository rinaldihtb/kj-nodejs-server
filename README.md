# kl-nodejs-server

NodeJs Framework with ExpressJS and Typescript

## Run Locally

Clone the project

```bash
  git clone git@github.com:rinaldihtb/kj-nodejs-server.git
```

Go to the project directory

```bash
  cd ks-nodejs-server
```

Install dependencies

```bash
  copy .env-sample .env
  npm install
```
Setup environtment

```bash
  copy .env-sample .env
```

Start the server

```bash
  npm start
```
